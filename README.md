# Tutor Platform (Web)

This is a full-stack web application where tutors can manage
courses, upload content, host live sessions, and interact
with students through quizzes and polls.

The focus of this project is backend APIs, data flow,
and scalable system design rather than UI polish.

## Current Status
Work in progress. Building core backend and web structure.

## Planned Features
- Tutor and student roles
- Course and content management
- Live session support
- Quizzes and polls
- Web-based dashboard

## Tech Stack (Planned)
- Backend: Node.js / Python
- Frontend: React
- Database: SQL
